# take-you-fly

## 问题
职业选手拥有庞大粉丝群，目前均是通过淘宝店、直播、录播视频等间接手段从粉丝群获取收益。

普通玩家渴望提高游戏水平，缺乏和偶像职业选手直接接触机会。

## 解决方案
提供一个平台，职业选手发布水友赛信息，普通玩家付费参与。

### GOGOSU
有教练和陪玩两种模式，均要付费。

教练模式有OB，天梯，普通匹配三种类型课程。

陪玩模式有按时，按天，按局，按月等形式。

进驻人群有主要是游戏高手，美女，少量主播。
